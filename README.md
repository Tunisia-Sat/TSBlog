# TSBlog
-------

TSBLog (Tunisia-Sat Blog) is an open source blog CMS Made in Tunisia ;-)

## How to Install it
--------

* copy & paste files to your server.  
* Change the database informations in config/db.php.  
* Import DataBase.sql to your database.  
* Enjoy & Develop!
